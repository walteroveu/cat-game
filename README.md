# 猫咪指认物品小游戏

一个有趣的猫咪指认物品小游戏，玩家需要根据猫咪的指令点击正确的物品。

## 功能特点

- 🐱 可爱的猫咪角色
- 🎯 简单有趣的游戏玩法
- 📱 支持移动设备
- 💾 可安装为PWA应用
- 🎵 有趣的动画和音效

## 游戏玩法

1. 猫咪会通过对话框提示你点击哪个物品
2. 点击正确的物品可以获得分数
3. 每次答对后，猫咪会抖动并显示"【蒸蚌】！"的提示

## 部署到GitHub Pages

1. 注册一个GitHub账号（如果没有）
2. 创建一个新的仓库，命名为"cat-game"
3. 上传本项目的所有文件到仓库
4. 在仓库设置中启用GitHub Pages：
   - 进入仓库 → Settings → Pages
   - 在"Source"下拉菜单中选择"main"分支
   - 点击"Save"按钮
5. 等待几分钟，GitHub会生成一个公开的URL
6. 在手机浏览器中访问该URL即可开始游戏

## 安装为PWA应用

1. 在手机浏览器中访问游戏URL
2. iOS设备：点击分享按钮 → 添加到主屏幕
3. Android设备：点击菜单按钮 → 添加到主屏幕
4. 游戏将作为应用安装到您的手机主屏幕

## 项目文件

- `index.html` - 游戏主页面
- `manifest.json` - PWA配置文件
- `service-worker.js` - 离线缓存服务
- `README.md` - 项目说明文档

## 技术栈

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- PWA技术

## 开发说明

如需修改游戏内容，可以编辑以下部分：

1. **添加新物品**：在`index.html`文件的`gameData.items`数组中添加新的物品对象
2. **修改动画效果**：在`<style>`标签内的`@keyframes`中修改动画定义
3. **调整游戏逻辑**：修改JavaScript部分的游戏逻辑函数
